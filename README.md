# Victor Fernandes — Full Stack Developer Portfolio

Portfolio cinematic e interativo construído com Next.js, React, Three.js e Tailwind CSS. Apresenta projetos, experiências e habilidades através de animações de scroll, efeitos 3D e um design system moderno.

**Live:** [portifolio-victorrony.vercel.app](https://portifolio-victorrony.vercel.app)

---

## Features

- **Hero Scroll Sequence** — animação frame-a-frame com 60 imagens geradas por IA (Nano Banana 2) sincronizada com o scroll do utilizador
- **Canvas 3D** — modelos Three.js com React Three Fiber (globo 3D no contacto, esferas de tecnologias)
- **Scroll Animations** — entradas com Framer Motion e GSAP, smooth scroll via Lenis
- **Bento Grid de Skills** — visualização das tecnologias e ferramentas com níveis de proficiência
- **Formulário de Contacto** — envio de email com Nodemailer e rate limiting
- **Performance** — imagens WebP (redução de 91.5%), lazy loading, bundle analyzer integrado
- **Design Responsivo** — mobile-first com Tailwind CSS

## Instalação

```bash
git clone https://github.com/victorrony/Portifolio.git
cd Portifolio
npm install
```

Crie um ficheiro `.env.local` com as suas credenciais de email:

```env
EMAIL_USER=seu@email.com
EMAIL_PASS=sua_password
```

```bash
npm run dev
```

---

## Scripts

```bash
npm run dev              # Servidor de desenvolvimento
npm run build            # Build de produção
npm run start            # Servidor de produção
npm run lint             # Linting
npm run format           # Formatar com Prettier
npm run optimize:images  # Otimizar imagens para WebP
npm run analyze          # Analisar bundle size
npm run test             # Testes com Vitest
npm run type-check       # Verificação de tipos TypeScript
```
