# 💼 Victor Fernandes - Full Stack Developer Portfolio

Um portfólio moderno e interativo construído com Next.js, React, Three.js e Tailwind CSS, apresentando projetos, experiências e habilidades através de animações 3D e design responsivo.

🌐 **Live Demo**: [https://portifolio-victorrony.vercel.app](https://portifolio-victorrony.vercel.app)

## ✨ Características

- **Animações Fluidas**: Transições suaves com Framer Motion
- **Gráficos 3D Interativos**: Modelos Three.js com React Three Fiber  
- **Design Responsivo**: Mobile-first com Tailwind CSS
- **Performance Otimizada**: Lazy loading, WebP images (-91.5%), Bundle 68.7KB
- **Formulário de Contato**: Com rate limiting e validação
- **Timeline Interativa**: Visualização de experiências profissionais

## 🏗️ Tecnologias

**Core**: Next.js 13, React 18, Tailwind CSS  
**3D & Animações**: Three.js, React Three Fiber, Framer Motion, GSAP  
**Backend**: Nodemailer, Rate limiting  
**Dev Tools**: sharp, bundle-analyzer, PropTypes

## 📦 Instalação

```bash
git clone https://github.com/victorrony/portifolio.git
cd portifolio
npm install
npm run dev
```

Configure `.env.local` com suas credenciais de email.

## 🚀 Scripts

```bash
npm run dev              # Desenvolvimento
npm run build            # Build produção  
npm run optimize:images  # Otimizar imagens para WebP
npm run analyze          # Analisar bundle
```
